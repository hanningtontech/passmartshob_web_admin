# Passmartshop Admin Dashboard Screenshots

This folder contains screenshots of all admin pages in the Passmartshop e-commerce platform.

## Screenshots Overview

### 1. Dashboard (17-07-58_7040.webp)
Main admin dashboard showing:
- Statistics cards (Total Categories, Products, Orders, Revenue)
- Quick Actions section
- System Status indicators
- Recent Activity feed

### 2. Categories List (17-08-08_7044.webp)
Category management page showing:
- All 10 predefined categories in table format
- Search functionality
- Edit and Delete buttons for each category
- Add Category button

### 3. Products List (17-08-15_7623.webp)
Product management page showing:
- Product listing with search and category filter
- Empty state (no products yet)
- Add Product button
- Table columns: Product Name, Category, Price, Stock, Featured, Actions

### 4. Add Product Form (17-08-23_5467.webp)
Comprehensive product creation form with:
- Basic Information section (Name, Description, Category)
- Pricing & Stock section (Price, Compare at Price, Stock Quantity)
- Product Flags (Mark as New, Featured Product, In Stock)
- Product Images section
- Product Variants section (below, not fully visible in screenshot)

### 5. Edit Product Form (17-08-29_8579.webp)
Similar to Add Product Form but for editing existing products

### 6. Add Category Form (17-08-39_5029.webp)
Category creation form with:
- Category Name
- Slug (auto-generated)
- Description
- Display Order
- Active checkbox

### 7. Product Types (17-08-49_9984.webp)
Product type management page showing:
- List of product types (Electronics, Appliances)
- Name, Description, Number of Fields, Status
- Edit, Delete, and Manage Fields buttons
- Add Product Type button

### 8. Import & Export (17-08-58_8959.webp)
Bulk import/export functionality showing:
- Import/Export toggle buttons
- Import Type selection (Products or Categories)
- Drag-and-drop file upload area
- Format guide
- Recent Imports section

## Design Notes

**Current Theme:**
- Dark background (navy/charcoal)
- Orange accent color (#FF6B35)
- Sidebar navigation
- Table-based layouts

**Key Features:**
- Responsive design
- Search and filter functionality
- CRUD operations for all entities
- Bulk import/export support
- Product variants management
- Custom product type fields
